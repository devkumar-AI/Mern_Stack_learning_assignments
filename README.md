## 🚀 About the Project
`bella_vista` is a clean, modern website designed for the food and hospitality industry. The goal of this initial assignment is to master semantic page structuring, styling layouts, and responsive design principles before introducing programming languages or databases.

### ✨ Current Features (Assignment 1)
* **Semantic Structure:** Built entirely with clean, accessible HTML5 elements.
* **Custom Styling:** Styled from scratch using CSS3 to create a unique visual aesthetic.
* **Layout Design:** Features structured sections utilizing modern CSS layout techniques (like Flexbox/Grid).
